# Aworker CLI
Noslate 命令行工具
## Usage
### invoke

`noslate invoke <options> [script.js]`

### invoke with debug

`noslate invoke --debug --data 'https://registry.npmmirror.com' ./examples/fetch.js`

`open chrome://inspect/`

### dev

`noslate dev [script.js]`

## install

install aworker

`noslate install <-f/--force>`
